## 简介
本项目是ASRT语音识别系统的Python3 SDK项目，并实现了一个调用样例。

## 安装
```shell
$ pip install asrt-sdk
```

或

```shell
$ python setup.py build
$ python setup.py install
```

## 软件环境要求
* 操作系统: 支持Windows 和 Linux、MacOS
* 开发环境： Python 3.6 +
